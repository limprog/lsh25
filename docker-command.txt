Перезапуск всех контейнеров:
docker compose restart
docker compose restart <имя_сервиса>

Переустановить все контейнеры:
cd /lsh25/yaml
docker compose down # Остановит и удалит все контейнеры, сети и тома (если они не именованы явно, как у вас)
docker compose up -d --build # Пересобирает образы и запускает все заново

Остановка контейнеров:
docker compose stop
docker compose stop <имя_сервиса>

Запуск контейнеров:
docker compose start
docker compose start <имя_сервиса>

Чистка незапущенных контейнеров и ресурсов:
docker system prune

Просмотр логов
docker compose logs -f python-app-users
